### Game of Life: No Border World
Those creatures live in a no border world ...

#### Features
* No border. top-botton connect & left-right connect
* Switch world size.
* Add random nodes.
* Auto generation. (constructing)

#### Simple Rule
* for `cell(i,j)` in `board(Row, Col)`
  - `i = i % Row`
  - `j = j % Col`
